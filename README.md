# LUMINA - Premium Lifestyle Store

This is a modern, responsive landing page for a lifestyle brand.

## File Organization
- `/index.html`: Main structure of the site.
- `/styles.css`: External stylesheet for all design and layouts.
- `/assets/`: local images here.
- `/screenshots/`: Desktop & Mbile View

Design Phase
The design process began with defining the brand identity and target audience. LUMINA is positioned as a premium lifestyle brand, so the design emphasizes quality, sophistication, and modernity. The color palette was chosen to reflect these values while maintaining accessibility.

Implementation
The HTML was written first, using semantic elements to structure the content. CSS was then added incrementally, starting with global styles and component styles, followed by responsive design adjustments.

Testing and Refinement
The page was tested across multiple browsers and devices to ensure consistent rendering and functionality. Responsive design was verified using browser DevTools and actual devices. Performance was optimized by removing unnecessary styles and consolidating similar rules.

Color Palette: Indigo primary color chosen for modern appeal and strong visual presence in e-commerce.

Typography: Clear hierarchy with large headlines, readable body text, and consistent sizing across breakpoints.

Layout: Responsive grid-based layout that adapts gracefully from desktop to mobile devices.

Interactions: CSS-based hover effects and transitions provide visual feedback without JavaScript overhead.

Accessibility: Semantic HTML, sufficient color contrast, and keyboard navigation support ensure inclusive design.
